# D4Nyos Chat
 
A sleek, lightweight web interface to interact with various AI models via the OpenRouter API. Built with vanilla HTML/CSS/JS for zero dependencies.
 
## Key Features
- Model selection from OpenRouter (free and paid)
- Image and document attachment support
- Dark/Light mode auto-switching
- Local storage for configuration
- Minimalist, responsive design
 
## Setup Instructions
1. Clone the repository. 2. Open `index.html` in any browser. 3. Enter your OpenRouter API key.
